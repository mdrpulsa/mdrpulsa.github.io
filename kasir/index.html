Ikuti langkah berikut:

  1. Buka template spreadsheet ini, lalu buat salinan (File → Buat salinan).

  3. Salin ID spreadsheet dari URL (deretan huruf/angka setelah /d/ sampai /edit).
  4. Tempel ID tersebut di bawah, lalu klik Simpan.
